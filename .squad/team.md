# Squad Team

## Project Context

- **Project:** EWSR_PMR_ModApp
- **What it does:** A mod manager for **Project Motor Racing**. Users drop a mod `.zip` and the app installs the files into the correct game locations. It also re-applies modded files that get reverted by game updates. Inspired by AMS2 Content Manager.
